Описание выложенных работ:

"TestServlets" - небольшое веб приложение с использованием сервлетов, сервлет-фильтров, базы данных, паттерна синглтон, jsp, JDBC.
содержит авторизацию, регистрацию пользователя и добавление товаров.

"DealsApp" - небольшое консольное приложение в которое можно записывать через консоль информацию о сделках(покупатель, продавец,товары)
В данном приложении применены принципы ООП.И имеется так же модуль сериализации файлов(товаров) в XML, например для отправки информации другому сервису.

"LiveWithDuck" - небольшое консольное приложение(задача собеседования) о утках, которые могут прилетать на озеро и улетать по команде пользователя, причем прилетая, утки начинают жить своей жизнью - плавают и крякают.
В данном приложении применен паттерн "Стратегия", создан дополнительный поток исполнения(чтобы утки могли независимо жить).
